Đặt file icon.ico của bạn vào thư mục này.
Inno Setup sẽ dùng file này làm icon cho file Setup.exe và icon Desktop.

Yêu cầu:
- Tên file: icon.ico
- Kích thước khuyến nghị: 256x256 px (multi-size ICO)
- Bạn có thể tạo file .ico từ PNG tại: https://convertico.com

Nếu không có file icon.ico, hãy xóa dòng SetupIconFile trong InterviewAI_Setup.iss.
